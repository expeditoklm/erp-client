Comment ça marche
1
Trouvez le code à 10 chiffres imprimé à côté du QR code sur l'emballage du produit.
2
Saisissez le code (format XXXXX-XXXXX) dans le champ ci-dessus.
3
Le système affiche l'image et le nom du produit original pour confirmation visuelle.
4
Chaque code ne peut être vérifié qu'une seule fois — si le vôtre a déjà été utilisé, le produit peut être une contrefaçon.
